# File Storage with Custom Path

Demonstrates creating a `FileStorage` instance with a custom directory and using it as the cache driver.

## Run

```bash
php index.php
```

## Expected Output

```
Cache directory: /path/to/08-file-storage-custom-path/my_cache
Stored value: stored in custom path
Cache files found: 1
Cleaned up.
```

Note: The directory path will reflect your local file system.
